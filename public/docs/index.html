Introduction

API Documentation for REST API Laravel Application

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://localhost/rest-laravel/public/

Authenticating requests

This API is authenticated by sending an Authorization header with the value "Bearer {YOUR_AUTH_TOKEN}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve {your_auth_token} by login into the application. See how to login in the Account Management documentation.

Account Management

APIs for managing users account

Create User

This endpoint lets you create a user.
Example request:
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://localhost/rest-laravel/public/api/register',
    [
        'headers' => [
            'Content-Type' => 'application/json',
        ],
        'json' => [
            'name' => 'Kevin Ringo',
            'email' => 'kevin@gmail.com',
            'phone' => 81234567890.0,
            'password' => '1234qweR',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
const url = new URL(
    "http://localhost/rest-laravel/public/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "Kevin Ringo",
    "email": "kevin@gmail.com",
    "phone": 81234567890,
    "password": "1234qweR"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Logout

Log the user out.

This endpoint lets you invalidate your token.
Example request:
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://localhost/rest-laravel/public/api/logout',
    [
        'headers' => [
            'Content-Type' => 'application/json',
        ],
        'query' => [
            'token'=> '{YOUR_AUTH_TOKEN}',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
const url = new URL(
    "http://localhost/rest-laravel/public/api/logout"
);

const params = {
    "token": "{YOUR_AUTH_TOKEN}",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "POST",
    headers,
}).then(response => response.json());

import json
